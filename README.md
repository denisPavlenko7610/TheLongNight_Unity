# The Long Night

The Long Night is a first-person survival game inspired by *The Long Dark*. Explore a harsh wilderness, gather resources, manage hunger, thirst, and fatigue, and survive against the elements.

## Features

* First-person exploration
* Inventory and item management
* Hunger, thirst, and health systems
* Campfire and fuel mechanics
* Interactive world objects
* Save/load system
* Modular gameplay architecture
* UI Toolkit based interface

## Technologies

* Unity 6
* C#
* VContainer
* Addressables (planing)
* UI Toolkit
* Scriptable Objects
* New Input System
* NUnit Tests (planing)
* SOLID principles and clean architecture
